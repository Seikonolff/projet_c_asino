# Projet Casino

Ce dépôt contient le projet Casino, un ensemble de jeux de casino implémentés en C.

## Auteurs

Norman ROUFFET
Lucca LAGOUARDE
Alexandre ATZENHOFFER
Mathis LABORY

## Description
Le projet inclut plusieurs jeux de casino classiques, tels que le blackjack, le poker, la roulette et les machines à sous. Il est écrit en C et utilise SQLite pour la gestion des données joueurs.

## Installation

Pour installer le projet :
    - Clonez le dépôt.
    - Exécutez make dans le répertoire du projet pour compiler.
    - Lancez l'exécutable généré pour démarrer le casino.
